# Dependencies

This is a complete command-oriented inventory for a new Linux installation.
Install every applicable group to enable every tracked feature. Distribution
package names differ; command names are the stable reference. The baseline
also assumes GNU/Linux `sh`, `bash`, GNU coreutils, findutils, grep, sed, awk,
util-linux (`setsid`, `lsblk`), procps (`pgrep`, `pkill`, `pidof`, `ps`),
`file`, and `sudo`.

## Templates and Scheduled Work

System samples are not installed automatically. Copy and adapt only after
installing the matching stack:

- `.local/share/sys-etc/portage/make.conf.template`: Portage and `emerge`
- `.local/share/sys-etc/systemd/network/wireless.network.template`:
  systemd-networkd and `networkctl`
- `.local/share/sys-etc/wpa_supplicant/wpa_supplicant.conf.template`:
  `wpa_supplicant`

Cron helpers require `cron`, `crontab`, `notify-send`, `xdotool`, `newsboat`,
and a running user DBus session. The example in `.local/bin/cron/README.md`
sets `DISPLAY` and `DBUS_SESSION_BUS_ADDRESS`, then sources `.profile`.
`checkup` uses `sudo apt update`; unattended use requires a narrowly scoped
passwordless sudo rule or a manual check.
