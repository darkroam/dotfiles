# 挂起项

这些项目有意不处于活动状态。仅在满足列出的恢复条件时重新开始；它们不是正常配置的缺陷。

## 状态与硬件

- [ ] 跨设备显示管理的设备适配器、manual marker、完整布局 health、多外屏/无盖桌面矩阵仍需
  明确恢复决定后再实施；通用范围见[显示管理设计](../project/display-management.md#未完成的通用工作)，
  各平台的恢复时间和验证条件见[平台档案索引](../platforms/index.md)。
- [ ] 仅在启用更多网络模块或当前两秒天气查询确实造成卡顿时，考虑异步 DWMBlocks 网络模块；
  不复制上游无限重试。
- [ ] 目标平台缺少标准背光能力时，先在对应平台档案验证可用后端，再统一 `xlight`、
  `xbacklight` 和状态栏动作；不得把一个设备的后端写成共享默认。
- [ ] 仅在暂停控制必须覆盖非 shell 启动的 MPV 时，加入全局 MPV IPC；使用小型跟踪包装器，
  不采用未经审查的上游 Lua 子模块。
- [ ] 仅在选择符号、维护数据源和可接受网络更新策略后，考虑市场行情模块。
